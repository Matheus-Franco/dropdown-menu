## :bulb: Clone

```bash
  # Clone this repository
  $ git clone https://github.com/Matheus-Franco/dropdown-menu.git

  # Go into the repository
  $ cd desafio-vilaApps

  # install all dependencies
  $ yarn install

  # Run
  $ yarn start
```
